## À propos

**Taaka** est une plateforme numérique nationale de publication et de recherche scientifique dédiée à la valorisation des travaux des chercheurs, enseignants-chercheurs et étudiants béninois.

Elle centralise, numérise et démocratise l'accès à la production scientifique nationale dans toutes les disciplines académiques reconnues par le **CAMES** (Conseil Africain et Malgache pour l'Enseignement Supérieur).

## Objectifs

- Développer une base de données numérique centralisée des travaux scientifiques béninois
- Offrir un espace sécurisé pour la publication en ligne d'articles, mémoires, thèses et rapports
- Mettre en place un moteur de recherche multicritère performant
- Favoriser la visibilité des chercheurs béninois à l'échelle nationale, régionale et internationale

## Disciplines couvertes

Les 8 Comités Techniques Spécialisés (CTS) officiels du CAMES :

| Code | Discipline |
|------|-----------|
| LSH | Lettres et Sciences Humaines |
| SEG | Sciences Économiques et de Gestion |
| SJP | Sciences Juridiques et Politiques |
| MPC | Mathématiques, Physique, Chimie |
| SNA | Sciences Naturelles et Agronomie |
| STI | Sciences et Techniques de l'Ingénieur |
| MHPOMVPA | Médecine, Pharmacie, Odontostomatologie, Médecine Vétérinaire |
| STAPS-JL | Sciences et Techniques des Activités Physiques, Sportives et Loisirs |

## Technologies utilisées

- **Frontend** : HTML5, CSS3, JavaScript (SPA)
- **Base de données** : Supabase (PostgreSQL)
- **Authentification** : Supabase Auth
- **Stockage PDF** : Supabase Storage
- **Emails** : Resend + Supabase Edge Functions
- **Hébergement** : GitHub Pages
